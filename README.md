# My Strava MCP Server

Learning project to build an MCP server that connects Claude to Strava.

## Setup

1. Install dependencies:
   \`\`\`bash
   npm install
   \`\`\`

2. Configure environment:
   \`\`\`bash
   cp .env.example .env
   # Edit .env with your Strava API credentials
   \`\`\`

3. Run in development:
   \`\`\`bash
   npm run dev
   \`\`\`

4. Build for production:
   \`\`\`bash
   npm run build
   npm start
   \`\`\`
